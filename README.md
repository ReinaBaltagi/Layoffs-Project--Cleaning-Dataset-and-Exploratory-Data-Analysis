# Layoffs Project - Data Cleaning and Exploratory Data Analysis

Overview
This project focuses on cleaning and analyzing a dataset about company layoffs using SQL. It was inspired by and follows Alex the Analyst's SQL tutorial series.

Tutorial Source: [Alex the Analyst](https://www.youtube.com/watch?v=OT1RErkfLNQ&list=PLvg9h-Cl5ZZ2_x2o14W3WFU1lTl7kRmCE&index=12)

Project Structure

Part 1: Data Cleaning
The dataset was cleaned following these steps:
1. Remove duplicates
2. Standardize the data
3. Handle null values and blank values
4. Remove unnecessary columns and rows

Part 2: Exploratory Data Analysis
After cleaning the dataset, I performed exploratory data analysis to:
- Get familiarized with the data
- Identify key trends and patterns
- Extract meaningful insights about layoffs

Skills Demonstrated
- SQL data cleaning techniques
- Handling duplicates and null values
- Data standardization
- Exploratory data analysis
- Data manipulation and querying
